# Canvas Engines Comparison

https://benchmarks.slaylines.io/

## Description

- [PixiJS](https://www.pixijs.com), [Two.js](https://two.js.org/), [Paper.js](http://paperjs.org/), [Fabric.js](http://fabricjs.com/) and [Pencil.js](https://pencil.js.org/);
- 1000, 2000 or 5000 different rectangles moving on canvas with different speed.

## Quick Start

```
$ yarn install
$ yarn build
$ yarn start
```
